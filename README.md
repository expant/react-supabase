# Poll Feed

Poll Feed — это удобное приложение, где вы можете создавать, просматривать опросы и голосовать за понравившиеся варианты.

## Возможности

- Просмотр ленты опросов
- Создание новых опросов
- Голосование в опросах
- Аутентификация пользователей
- Управление сессией пользователя

## Стек технологий

- React
- TypeScript
- Supabase (BaaS: хранение данных, аутентификация, real-time взаимодействие)
- Vite
